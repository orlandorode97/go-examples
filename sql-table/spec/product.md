# Mini MySQL Client

## Goal
A lightweight client application written in Go to connect to a MySQL server and execute SQL queries through a minimal UI.

## Principles
- Minimal UI
- Fast startup
- No heavy frameworks
- Focus on query execution

## Core Features
1. Connect to a MySQL server
2. List databases
3. List tables in a database
4. Execute SQL queries
5. Display query results

## Non Goals
- Schema visual editor
- Query builder
- Migration management
- ORM features
